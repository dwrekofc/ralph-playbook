## Grading Scale

- **Pass:** Feature works as specified. All success criteria met.
- **Partial:** Some criteria met. Feature partially functional. Specific gaps documented.
- **Fail:** Feature broken, missing, or does not meet any success criteria.

## Pass Rate Calculation

```
pass_rate = (features_pass / features_total) * 100
```

Partial scores count as 0.5 for pass rate calculation.
