## Build & Run

- Runtime: Bun
- UI: shadcn/ui with Base UI primitives + Tailwind CSS
- Language: TypeScript (strict)
- Build: `bun run build`
- Dev: `bun run dev`

## Validation

- Tests: `bun test`
- Typecheck: `bun run tsc --noEmit`
- Lint: `bunx biome check .`
- Format: `bunx biome format . --write`
